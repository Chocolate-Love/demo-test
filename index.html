다층 상식 통합 예제 Demo Page

이 페이지는 GitHub Pages에 올릴 수 있는 정적 프론트엔드 예시입니다. 실제 비디오 처리와 요약은 Docker 서버 API가 담당하고, 여기서는 결과만 보기 좋게 렌더링합니다.

입력

파일 업로드 또는 영상 URL
영상 파일 드래그 앤 드롭

또는 아래 버튼으로 파일을 선택하세요. 현재는 데모 UI이며, 실제 업로드는 Docker 백엔드 API와 연결하면 됩니다.

선택된 파일이 없습니다.
백엔드 연결 전 상태입니다. 예시 결과를 먼저 확인할 수 있습니다.
연결 포인트 GitHub Pages에서는 서버 코드를 실행하지 않으므로, 프론트는 fetch()로 Docker 서버의 API에 요청해야 합니다. 예시는 아래와 같습니다.

POST /api/summarize
GET /api/job/:id
